Step one: HTML

You may want to create text but also separated boxes or a list of products etc...
All of this can be done with html elements which always start and end with < and >.
for example to create this text I am using <p> which is the element for a paragraph. As it would take too long to explain all the possible elements, here is a link to a page listing all of them.

Now that we have that, how to create an html page from scratch???
Well, it is easy (easy way at the end) :

  1. start with <!Doctype html>
  2. create <head> in which you declare the next things
  3. In the head, declare <meta> with your viewpport and the fact you use utf-8, the title element with the name of the website, and, all your links to css files (we will come back to css in the next chapter)
  4. create body under head in which you'll put a header element and a footer element.
  5. The header contains the menu
  6. outside in the middle of head and footer is your website
  7. The footer contains the credit block of your page
  8. THE EASY WAY : at the start when there is still nothing, put ! and you will immediately have a guess from the code app, if you click tab, everything will appear (apart from your css link), just have to fill it
Step two: CSS

Now you have words appearing in front of you yes ! but... aren't they just boring, pitch black on white... I know ! let's add css, "cascade style sheet". CSS is a mask which you give your website to hide it's horrible face, to declare a stylesheet in html use
<link rel="stylesheet" href="location_of_the_stylesheet">

All the properties can be found here, but there is some properties which are to be used way more than others :

  • a command to do a list without decorations in a single line -> text-decoration : none;
  • a command to do margins from the outside of the element (think of it as pushing the car while being outside of the car) -> margin-right/top/left/bottom : 0px/0%
  • a command to do margins from the inside of the element (think of it as pushing an object in the car while being in the car) -> padding-right/top/left/bottom : 0px/0%
  • a command to move to the end of your div -> float : right(or left);
  • a command to have the element disconnect from the listed order (being in spectator in minecraft and going everywhere without having to move all the elements in front of you) -> position : relative; (this is one of the best)
  • a command to choose if your element will appear or not, if it will be inline or not etc... -> display : none/inline-block etc...;
This is how a page is usually built, you can see the use of padding and margin
